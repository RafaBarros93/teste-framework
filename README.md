# Teste Framework



Desafio para vaga desenvolvedor Node.js developing in [node](https://nodejs.org/en/).

## Installation

Use the package manager [yarn](https://classic.yarnpkg.com/) to install.

## Install dpendencies

```bash
yarn
```

## Run application

```bash
yarn start
```

## Testes

```bash
#integração
yarn test:integration

#unitário
yarn test:unit

#ci
yarn test:ci

#staged
yarn test:staged


```
